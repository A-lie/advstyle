# 更新说明

## 版本 1.1.0 - 属性面板优化

### 主要改进

#### 1. 属性面板布局优化
- **面板宽度调整**：从300px增加到350px，提供更多显示空间
- **标签宽度优化**：从80px调整为70px，为控件留出更多空间
- **间距调整**：优化各元素间距，提升视觉效果

#### 2. 交互方式改进
- **滑动条替代数字输入框**：所有数值参数（坐标、尺寸、透明度等）改为滑动条形式
- **实时输入支持**：滑动条支持直接输入数值，提供更精确的控制
- **范围限制**：为不同属性设置合理的取值范围

#### 3. 具体优化内容

##### 位置和尺寸控制
- **X/Y坐标**：使用滑动条，范围根据画布大小和元素尺寸动态调整
- **宽度/高度**：使用滑动条，最小值50px，最大值2000px
- **智能范围**：坐标最大值会根据元素尺寸自动调整，防止元素超出画布

##### 样式属性
- **透明度**：滑动条范围0-255，显示百分比值
- **层级**：滑动条范围0-100
- **字体大小**：滑动条范围12-200px（文字组件）

##### 时间间隔
- **图片切换间隔**：滑动条范围500-60000毫秒
- **文字切换间隔**：滑动条范围500-60000毫秒

#### 4. 视觉改进
- **统一样式**：所有属性组件采用一致的设计风格
- **清晰标签**：每个滑动条都有明确的标签说明
- **分组布局**：相关属性进行分组显示，提升可读性

### 技术实现

#### 滑动条配置
```vue
<el-slider
  v-model="localElement.x"
  :min="0"
  :max="getMaxX"
  :step="1"
  @change="updateElement"
  show-input
  :show-input-controls="false"
  input-size="mini"
></el-slider>
```

#### 动态范围计算
```javascript
computed: {
  getMaxX() {
    return Math.max(1920 - this.localElement.width, 0)
  },
  getMaxY() {
    return Math.max(1080 - this.localElement.height, 0)
  }
}
```

## 使用说明

### 新的操作方式

1. **滑动调整**：拖拽滑块快速调整数值
2. **精确输入**：点击输入框直接输入精确数值
3. **键盘控制**：使用方向键进行微调
4. **范围提示**：滑动条显示可用的数值范围

### 注意事项

- 坐标范围会根据元素尺寸自动调整
- 透明度显示为百分比，内部仍使用0-255范围
- 所有数值变化都会实时反映到画布上

# 广告设计系统演示指南

## 访问地址

启动开发服务器后，访问以下地址：
- 首页：http://localhost:8080
- 设计器：http://localhost:8080/designer

## 演示步骤

### 1. 进入设计器
1. 在首页点击"开始设计"按钮
2. 或直接访问 http://localhost:8080/designer

### 2. 设置画布尺寸
1. 在左侧"屏幕尺寸"区域设置画布大小
2. 可以使用预设尺寸（如1920x1080）或自定义尺寸
3. 画布会自动缩放以适应编辑区域

### 3. 添加图片组件
1. 从左侧组件库拖拽"图片"组件到画布
2. 系统会自动弹出文件选择对话框
3. 选择一张或多张图片文件
4. 图片会显示在画布上，可以拖拽移动和调整大小

### 4. 设置图片属性
1. 点击选中图片组件
2. 在右侧属性面板中可以设置：
   - 位置和尺寸
   - 显示方式（等比例居中/拉伸填充）
   - 背景颜色和透明度
   - 如果有多张图片，可设置切换间隔

### 5. 添加视频组件
1. 拖拽"视频"组件到画布
2. 选择视频文件
3. 在属性面板中设置播放选项：
   - 自动播放
   - 循环播放
   - 显示方式

### 6. 添加文字组件
1. 拖拽"文字"组件到画布
2. 双击文字组件可直接编辑文本内容
3. 在属性面板中设置：
   - 字体大小、颜色、加粗
   - 对齐方式（左对齐、居中、右对齐、滚动）
   - 如果选择滚动，可设置滚动速度
   - 可添加多个文本内容实现轮播

### 7. 添加容器组件
1. 拖拽"容器"组件到画布
2. 可作为其他元素的背景
3. 设置背景颜色和透明度

### 8. 调整元素
- **移动**：点击并拖拽元素
- **调整大小**：选中元素后拖拽四角的调整手柄
- **删除**：选中元素后点击右上角的删除按钮
- **层级**：在属性面板中调整zIndex值

### 9. 预览效果
1. 点击顶部"预览"按钮
2. 在弹出的对话框中查看最终效果
3. 图片轮播、文字滚动等动画效果会正常播放

### 10. 保存和下发
1. 点击"保存设计"将配置保存到浏览器本地存储
2. 点击"下发到设备"可模拟推送到广告终端
3. 输入设备SN号，系统会生成JSON配置数据

## 示例场景

### 场景1：商店广告屏
1. 设置画布为1920x1080
2. 添加背景图片（商店logo或背景）
3. 添加产品图片轮播（3-5张产品图）
4. 添加滚动文字显示促销信息
5. 设置合适的切换间隔和滚动速度

### 场景2：信息发布屏
1. 设置画布为1366x768
2. 添加标题文字（居中显示）
3. 添加内容区域（多个文本轮播）
4. 添加底部滚动字幕
5. 设置不同的文字样式和颜色

### 场景3：视频广告
1. 设置画布为1080x1920（竖屏）
2. 添加主视频（自动播放、循环）
3. 添加透明背景的文字覆盖层
4. 设置文字滚动或轮播效果

## 技术要点

### 支持的文件格式
- **图片**：JPG, PNG, GIF, WebP
- **视频**：MP4, WebM, OGV

### 性能建议
- 图片建议压缩后使用，单张不超过2MB
- 视频建议使用H.264编码，时长不超过30秒
- 同时显示的元素建议不超过10个

### 浏览器要求
- 建议使用Chrome、Firefox、Safari等现代浏览器
- 需要支持HTML5和ES6特性
- 需要支持File API和Canvas API

## 故障排除

### 常见问题
1. **图片不显示**：检查文件格式和大小
2. **视频无法播放**：检查浏览器是否支持该视频格式
3. **拖拽不响应**：刷新页面重试
4. **保存失败**：检查浏览器是否支持localStorage

### 调试方法
1. 打开浏览器开发者工具
2. 查看Console面板的错误信息
3. 检查Network面板的资源加载情况
4. 查看Application面板的localStorage数据

# 动画效果使用指南

## 概述
广告编排设计器现在支持在预览模式下显示各种动画效果，包括文字滚动、图片轮播、视频播放等。

## 支持的动画效果

### 1. 文字滚动动画
- **设置方法**：
  1. 添加文字组件
  2. 在右侧属性面板中，将"对齐方式"设置为"滚动"
  3. 调整"滚动速度"（1-9级，数值越大速度越快）
  4. 点击"预览"按钮查看滚动效果

- **效果说明**：文字会从右侧进入，向左滚动，然后从左侧退出，循环播放

### 2. 图片轮播动画
- **设置方法**：
  1. 添加图片组件
  2. 在属性面板中点击"添加图片"，上传多张图片
  3. 设置"切换间隔"（毫秒为单位）
  4. 点击"预览"查看轮播效果

- **效果说明**：多张图片会按设定的时间间隔自动切换

### 3. 文字轮播动画
- **设置方法**：
  1. 添加文字组件
  2. 在属性面板中点击"添加文本"，添加多段文字内容
  3. 设置"切换间隔"（毫秒为单位）
  4. 点击"预览"查看轮播效果

- **效果说明**：多段文字会按设定的时间间隔自动切换显示

### 4. 视频播放
- **设置方法**：
  1. 添加视频组件
  2. 选择视频文件
  3. 在属性面板中勾选"自动播放"和"循环播放"
  4. 点击"预览"查看播放效果

- **效果说明**：视频会在预览模式下自动播放（静音）

## 预览功能

### 如何预览
1. 在设计器中添加和配置元素
2. 点击顶部的"预览"按钮
3. 在弹出的预览对话框中查看动画效果

### 预览特点
- **实时动画**：所有动画效果都会在预览中实时显示
- **真实效果**：预览效果与最终播放效果一致
- **无干扰**：预览模式下不显示编辑控件，专注于内容展示

## 层级管理

### 层级控制
- 每个元素都有独立的层级（zIndex）
- 层级数值越大，元素显示越靠前
- 可以通过属性面板的滑块精确设置层级
- 也可以使用快捷按钮：置顶、上移、下移、置底

### 层级面板
- 左侧组件库下方显示所有元素的层级关系
- 按层级从高到低排序显示
- 点击元素可以选中，支持快速层级调整

## 使用技巧

### 1. 文字滚动优化
- 较短的文字适合较快的滚动速度
- 较长的文字建议使用较慢的滚动速度
- 可以通过调整元素宽度来控制滚动区域

### 2. 轮播时间设置
- 图片轮播建议间隔3-5秒
- 文字轮播建议间隔2-4秒
- 根据内容长度调整合适的显示时间

### 3. 层级规划
- 背景元素使用较低的层级
- 重要内容使用较高的层级
- 利用层级面板直观管理元素关系

### 4. 预览测试
- 经常使用预览功能检查效果
- 注意动画的流畅性和协调性
- 确保所有动画都能正常播放

## 注意事项

1. **性能考虑**：过多的动画元素可能影响性能
2. **浏览器兼容**：某些老旧浏览器可能不支持所有动画效果
3. **文件大小**：视频文件较大时加载可能较慢
4. **自动播放**：某些浏览器可能阻止视频自动播放

## 故障排除

### 动画不显示
1. 检查是否在预览模式下查看
2. 确认动画参数设置正确
3. 检查浏览器是否支持CSS动画

### 视频不播放
1. 确认视频格式是否支持
2. 检查是否勾选了"自动播放"
3. 某些浏览器需要用户交互才能播放视频

### 轮播不工作
1. 确认是否添加了多个内容项
2. 检查切换间隔是否设置正确
3. 确保在预览模式下查看

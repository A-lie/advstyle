# 文本滚动功能测试指南

## 测试步骤

### 1. 创建滚动文本
1. 打开设计器页面：http://localhost:8080/#/designer
2. 从左侧组件库拖拽"文字"组件到画布上
3. 选中文字组件，在右侧属性面板中：
   - 修改文本内容为较长的文字（如："这是一段很长的滚动文字，用来测试滚动动画效果"）
   - 将"对齐方式"设置为"滚动"
   - 调整"滚动速度"（1-9级）

### 2. 验证编辑模式
- 在编辑模式下，文字应该显示为静态文本
- 可以看到滚动速度设置选项

### 3. 验证预览模式
1. 点击顶部的"预览"按钮
2. 在预览对话框中，文字应该：
   - 从右侧进入画布
   - 向左滚动移动
   - 从左侧退出画布
   - 循环重复

### 4. 测试不同速度
- 速度1：最慢滚动
- 速度5：中等速度
- 速度9：最快滚动

## 故障排除

### 如果滚动不工作：

1. **检查设置**：
   - 确认对齐方式已设置为"滚动"
   - 确认在预览模式下查看

2. **检查文本长度**：
   - 文本太短可能看不出滚动效果
   - 建议使用较长的文本进行测试

3. **检查元素尺寸**：
   - 确保文本元素有足够的宽度
   - 文本元素不能太窄

4. **浏览器兼容性**：
   - 确保浏览器支持CSS动画
   - 尝试刷新页面

### 预期效果：
- 文字从右侧100%位置开始
- 平滑移动到左侧-100%位置
- 动画持续时间根据速度调整
- 无限循环播放

## 技术细节

### CSS动画：
```css
@keyframes scrollText {
  0% { transform: translateX(100%); }
  100% { transform: translateX(-100%); }
}
```

### 动画持续时间计算：
- 速度1：11秒
- 速度5：7秒  
- 速度9：3秒

### 样式应用：
- `animation: scrollText ${duration} linear infinite`
- `white-space: nowrap`
- `display: inline-block`
